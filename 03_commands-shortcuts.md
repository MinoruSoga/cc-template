# Claude Code コマンド・ショートカット一覧

## スラッシュコマンド

### 基本操作

| コマンド | 説明 |
|---------|------|
| `/help` | ヘルプを表示 |
| `/clear` | 会話をクリア |
| `/compact` | 会話を圧縮（トークン節約） |
| `/status` | システム・アカウント状態を表示 |
| `/usage` | 使用量を表示 |
| `/cost` | セッションのコストを表示 |

### 会話管理

| コマンド | 説明 |
|---------|------|
| `/resume` | 過去の会話を再開 |
| `/export` | 会話をエクスポート |
| `/rewind` | 会話を巻き戻し（コード変更を元に戻す） |

### 設定・カスタマイズ

| コマンド | 説明 |
|---------|------|
| `/config` | 設定を表示・変更 |
| `/model` | モデルを変更（Opus, Sonnet, Haiku） |
| `/permissions` | 権限設定を管理 |
| `/vim` | Vim モードの切り替え |
| `/theme` | テーマの変更 |

### MCP 関連

| コマンド | 説明 |
|---------|------|
| `/mcp` | MCP サーバー一覧・管理 |
| `/mcp add` | MCP サーバーを追加 |

### プロジェクト管理

| コマンド | 説明 |
|---------|------|
| `/add-dir` | 作業ディレクトリを追加 |
| `/memory` | メモリファイルを編集 |
| `/context` | コンテキスト情報を表示 |

### ツール・機能

| コマンド | 説明 |
|---------|------|
| `/doctor` | 問題診断 |
| `/terminal-setup` | ターミナル設定 |
| `/statusline` | ステータスライン設定 |
| `/agents` | カスタムエージェント管理 |
| `/hooks` | フック管理 |
| `/plugin` | プラグイン管理 |
| `/todos` | Todo リストを表示 |

### その他

| コマンド | 説明 |
|---------|------|
| `/release-notes` | リリースノートを表示 |
| `/upgrade` | プランのアップグレード |

## キーボードショートカット

### 入力・編集

| ショートカット | 説明 |
|---------------|------|
| `Enter` | メッセージ送信（単一行） |
| `Shift + Enter` | 改行（複数行入力） |
| `Tab` | ファイル名オートコンプリート |
| `Shift + Tab` | Auto-accept モード切り替え |
| `Ctrl + G` | 外部エディタでプロンプト編集 |
| `Ctrl + U` | 入力を取り消し |
| `Ctrl + Y` | 削除したテキストを貼り付け |

### 会話操作

| ショートカット | 説明 |
|---------------|------|
| `Esc` | 実行を中断 |
| `Esc Esc` | 即座に中断（Auto-accept 時） |
| `Ctrl + R` | 履歴検索 |
| `Ctrl + O` | トランスクリプトモード |
| `Ctrl + B` | Bash コマンドをバックグラウンド実行 |

### モード切り替え

| ショートカット | 説明 |
|---------------|------|
| `Tab` | thinking モード切り替え |
| `Alt + M` | モード切り替え（Windows） |

### Vim モード（有効時）

| ショートカット | 説明 |
|---------------|------|
| `j/k` | メニュー上下移動 |
| `h/l` | 左右移動 |
| `u` | 取り消し |
| `c` | 変更 |
| `f/F` | 文字検索（前方/後方） |
| `t/T` | 文字まで移動 |

### Emacs/Bash スタイル

| ショートカット | 説明 |
|---------------|------|
| `Ctrl + N` | 次の項目 |
| `Ctrl + P` | 前の項目 |
| `Ctrl + A` | 行頭へ |
| `Ctrl + E` | 行末へ |

## 特殊入力

### メモリ追加

```
# <メモリに追加したい内容>
```

`#` で始めるとメモリに追加される。

### バックグラウンドタスク（Web版）

```
& <タスク内容>
```

`&` で始めるとバックグラウンドタスクとして実行。

### @メンション

| 記法 | 説明 |
|-----|------|
| `@filename` | ファイルをコンテキストに追加 |
| `@folder/` | フォルダをコンテキストに追加 |
| `@agent-name` | カスタムエージェントを呼び出し |
| `@mcp-server` | MCP サーバーを指定 |

### Thinking モード

プロンプトに以下のキーワードを含めると thinking モードが有効に：

- `think`
- `think harder`
- `ultrathink`

## CLI オプション

```bash
# 基本的な起動
claude

# 特定のプロンプトで起動
claude "質問内容"

# 前回の会話を継続
claude --continue
claude -c

# 会話を再開
claude --resume

# 非対話モード（出力のみ）
claude -p "質問"

# 追加ディレクトリを指定
claude --add-dir /path/to/dir

# MCP 設定ファイルを指定
claude --mcp-config config.json

# システムプロンプトを追加
claude --append-system-prompt "追加指示"

# デバッグモード
claude --debug
```
