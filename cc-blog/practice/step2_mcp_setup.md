# ステップ 2: MCP とツールのセットアップ

関連ブログ: `mcp-and-tools.md`

## 1. プロジェクト MCP サーバーの有効化

`Template/.claude/settings.json` で、以下の設定が `true` になっていることを確認してください：

```json
"enableAllProjectMcpServers": true
```

これにより、Claude はローカルの `.mcp.json` ファイルを信頼するようになります。

## 2. Cipher MCP (メモリエージェント)

このテンプレートには、ローカルメモリエージェントである **Cipher** が統合されています。

**設定手順**:

1.  `Template/cipher.yml` をプロジェクトのルートにコピーします。
2.  `.mcp.json` を更新し、そのファイルの絶対パスを指定します。

**目的**:
Cipher を使用することで、`context-and-skills.md` で説明されている「構造化ノートテイキング / エージェントメモリ」パターンを実装できます。これにより、コンテキストリセット後も記憶を保持できます。

## 3. Playwright とブラウザツール

テンプレートは `playwright` と `chrome-devtools` を参照しています。これらによりエージェントは以下が可能になります：

- スクリーンショットの撮影（視覚的フィードバック）。
- Web アプリのデバッグ。

**実践**:
`.mcp.json` で設定されている必要な MCP サーバーがインストールまたは構成されていることを確認してください。
